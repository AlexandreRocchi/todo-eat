# todo-eat

**todo-eat** est une application de gestion de liste de courses simple et intuitive, conçue pour vous aider à organiser vos achats alimentaires de manière efficace.

## Fonctionnalités

- **Ajout d'articles** : Ajoutez de nouveaux articles à votre liste de courses avec un nom, une quantité et une unité de mesure (grammes, litres ou pièces).
- **Modification d'articles** : Mettez à jour les détails des articles existants dans votre liste.
- **Suppression d'articles** : Supprimez les articles dont vous n'avez plus besoin.
- **Affichage de la liste** : Consultez la liste complète des articles à acheter.
- **Enregistrement de modèles** : Sauvegardez des listes de courses comme modèles pour les réutiliser plus tard.
- **Chargement de la semaine précédente** : Récupérez la liste de courses de la semaine précédente pour gagner du temps.
- **Accès aux templates** : Naviguez vers une page de modèles préenregistrés pour faciliter la création de listes.
- **Paramètres personnalisés** : Accédez aux options de configuration pour personnaliser votre expérience.
- **Stockage local** : Les données de votre liste sont stockées localement sur votre appareil pour une meilleure confidentialité.
- **Import/Export de JSON** : Exportez ou importez vos listes de courses sous forme de fichiers JSON pour une sauvegarde ou un partage facile.

## Installation

1. **Cloner le dépôt** :

   ```bash
   git clone https://github.com/AlexandreRocchi/todo-eat.git
   ```

2. **Naviguer dans le répertoire du projet** :

   ```bash
   cd todo-eat
   ```

3. **Ouvrir le fichier `index.html`** dans votre navigateur pour utiliser l'application en local.
